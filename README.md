# Enhanced ADX Market Data Dashboard

## Overview
The Enhanced ADX Market Data Dashboard provides users with an interactive platform to explore the ADX Market Bulletin dataset. With real-time updates, dynamic visualizations, and user-friendly features, the dashboard empowers financial analysts, investors, policymakers, and researchers to make informed decisions based on comprehensive financial data.

## Features
- Real-Time Data Updates
- Interactive Visualizations
- Customizable Reports
- Multilingual Support (English and Arabic)
- User-Friendly Interface
- Data Export Options

## Prerequisites
1. Python 3.8 or higher.
2. Libraries: `pandas`, `plotly`, and `dash`.
3. Internet connection to download the dataset.

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/Enhanced-ADX-Dashboard.git
   cd Enhanced-ADX-Dashboard
   
2. Install required packages:
   bash
   pip install -r requirements.txt
   
3. Run the application:
   bash
   python app.py
   
4. Open your browser and navigate to `http://127.0.0.1:8050/` to access the dashboard.

## Usage
1. Select a sector from the dropdown menu to filter the data.
2. View the sector performance chart to analyze trading volumes and last close prices.
3. Explore the market capitalization distribution pie chart.
4. Export the data for further analysis as needed.

## Dataset
The dashboard utilizes the "ADX Market Bulletin - (30th April 2026)" dataset, which is available at: [Download Dataset](https://opendata.abudhabi.ae/finance/adx-market-bulletin-30-04-2026.xlsx).
